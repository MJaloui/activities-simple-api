
# Activities to Cure Boredom

**Project Overview**

This is a web application that fetches and displays random activity suggestions from the Bored API. When users are looking for something to do, they can click a button to receive a new activity idea, complete with details like activity type, number of participants, and accessibility. A simple tool to help cure boredom with endless activity suggestions.

**Core Features**

* Fetch random activities from the Bored API
* Display activity suggestions with relevant details
* Button to generate new activity ideas
* Clean, user-friendly interface

**Technologies**

* **HTML:** Structure and content display
* **CSS:** Styling and visual presentation
* **JavaScript:** API calls with Fetch and DOM manipulation
* **Bored API:** External API for activity data

**Key Takeaways**

* Working with **external APIs** and fetching data
* Using the **Fetch API** for asynchronous requests
* Handling **JSON responses** from APIs
* Implementing **DOM manipulation** to display dynamic content
* Creating **event-driven interactions** for user engagement
